Чтобы запустить приложение- в виртуальном web сервере перейти на url localhost.
Чтобы запустить приложение, нужно открыть в браузере Index.php. 
Нажать на кнопку GLitch Em. После прохождения процесса Гличевания, если хочется еще рандомных вариантов аватаров- следует перейти назад( на страницу с  кнопкой) и еще раз нажать на неё.
Из-за недоработанности тестового задания, не канает получать прогличенные аватарки просто- обновляя страницу.
Количество отображаемых картинок на страинце определяется параметром count (в test.php -   'count'=>40)
